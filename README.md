# Finance Management Application

Flask-based web application for personal finance management with authentication, transaction tracking, and financial statistics.

## Features

- User authentication and authorization
- Transaction management (income/expense)
- Category management  
- Financial dashboard and statistics
- Data isolation between users

## Technology Stack

- Python 3.14
- Flask 2.x
- SQLite database
- JWT authentication
- Pytest for testing
